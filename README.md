🧬 Praticando Java: Herança, Polimorfismo e Interfaces
Este repositório reúne os desafios e projetos práticos desenvolvidos durante o curso de Java: Herança, Polimorfismo e Interfaces na Alura. O objetivo principal foi entender como criar hierarquias de classes inteligentes e sistemas que podem crescer sem se tornarem "rígidos" ou repetitivos.

🚀 O que aprendi neste módulo?
🧱 Herança (extends)
Aprendi a evitar a duplicação de código criando classes "Mãe" que compartilham atributos e comportamentos com classes "Filhas".

Uso da palavra-chave super para acessar membros da classe superior.

Diferença entre Classes Comuns e Classes Abstratas (que servem apenas como molde e não podem ser instanciadas).

🎭 Polimorfismo
O conceito de "muitas formas". Entendi como tratar objetos diferentes de maneira genérica, permitindo que o sistema seja muito mais dinâmico.

Uso do @Override para sobrescrever métodos e dar comportamentos específicos para cada subclasse.

Referências de tipos genéricos apontando para objetos específicos.

📑 Interfaces (implements)
A criação de "contratos" de comportamento. Com interfaces, aprendi a garantir que uma classe execute certas ações, independentemente de quem ela herda.

Desacoplamento de código.

Como uma classe pode implementar múltiplas interfaces (diferente da herança simples).

🛠️ Tecnologias e Conceitos
Java JDK 17+

Orientação a Objetos (OO)

Clean Code: Escrevendo métodos que fazem sentido na hierarquia.

"Herança permite que você reutilize o que já existe; Polimorfismo permite que você substitua o que já existe; Interfaces permitem que você defina o que deve existir." 🚀
